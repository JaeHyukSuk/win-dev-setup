# win-dev-setup

Simple Windows bootstrap script for common developer tools.

## Included

- Git
- GitHub CLI
- Node.js LTS
- Python 3.12
- FFmpeg
- Optional CLI tool via npm

## Usage

Run [`windows_dev_tools_setup.bat`](/C:/Users/Administrator/win-dev-setup/windows_dev_tools_setup.bat) in a normal Command Prompt or PowerShell session.

If a tool is installed but not immediately available, open a new terminal window and try again.
